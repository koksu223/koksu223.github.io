# Projekt międzykierunkowy — strona HTML

Krótka instrukcja publikacji na GitHub Pages:

1. Zainicjalizuj repozytorium git lokalnie:

```
git init
git add .
git commit -m "Initial commit"
git branch -M main
```

2. Utwórz repozytorium na GitHub (przez stronę lub `gh repo create`) i dodaj remote:

```
git remote add origin https://github.com/<twoj-uzytkownik>/<nazwa-repo>.git
git push -u origin main
```

3. W ustawieniach repo na GitHub włącz GitHub Pages (branch `main`, folder `/`) lub skonfiguruj branch `gh-pages`.

Jeśli chcesz, mogę pomóc utworzyć repo i wypchnąć zmiany (potrzebny będzie dostęp do `gh` CLI lub token).
